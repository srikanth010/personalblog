# personalblog


---

## 🛠️ Build & Run Instructions (Backend)

### ✅ Prerequisites
- Java 17+ installed
- Maven installed (`mvn -v` to confirm)

---

### 🔨 Build the Project

From the **root folder** of the project (where `pom.xml` is located), run:

```bash
mvn clean install
cd target
java -jar personalblog-0.0.1-SNAPSHOT.jar
